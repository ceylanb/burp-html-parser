# Burp HTML Parser

This is a Burp Extension to apply CSS selectors
to HTML content from the HTTP message viewer. Inspired by [Burp-JQ](https://github.com/synacktiv/burp-jq).

![Demo GIF](img/demo.gif)

## Build

```bash
$ gradle fatJar
```

## Credits

Burp HTML Parser relies on `jsoup`.
